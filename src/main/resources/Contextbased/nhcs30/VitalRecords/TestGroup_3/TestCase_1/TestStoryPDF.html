Description

When the jurisdictional vital records office electronically sends the HL7 ADT^A04 message containing cause of death literal and demographic information to the national statistical agency (NCHS), the message is automatically parsed to create demographic and cause of death literal files in a format that is required by the National Vital Statistic System's automated cause of death coding system. This automated system translates the text literal cause of death terms, reported by the jurisdiction for Madeline Smith's cause of death, into ICD-10 codes. This system also selects an underlying cause of death (which is also reported as an ICD-10 code).

If the National Vital Statistic System's automated cause of death coding system cannot successfully code the decedent's cause of death record, the record is manually coded at NCHS. Madeline Smith's record needed to be manually coded due to some errors in the jurisdictional reporting of duration relating to the cause of death text literal terms on the record. As a result, the data values for "Intentional Reject Edit Flag" and "Manual Underlying Cause of Death" were included in the coded cause of death record to be returned to the jurisdiction. 

Once the coded cause of death record is complete, NCHS electronically sends the HL7 ADT^A04 message containing Ms. Smith's coded cause of death, and other related information, back to the jurisdiction.
